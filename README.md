### Setup

To clone this repository to your local machine, open a terminal and select desired folder:
```sh
  cd JavaScriptCapstone
```
Clone:
```sh
  git clone https://github.com/Mar12358/JavaScriptCapstone
```


### Install
- To install the project requirements:
```sh
  npm install
```


### Usage
- To visualize the project, run:
```sh
  npm start
```
